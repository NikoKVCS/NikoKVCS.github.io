Git 的使用笔记

 2017/11/23

拷贝的步骤

1. git clone xxx

2. cd .\android\

a. 进入到此本地文件仓库

b. 这一步是关键的, 因为有时候你有好多个项目, 例如 github 文件夹下, 有 /android/.. 也有 /forcestorm/.. 还有/python-three/..

c. 所以切换到文件进去, 他才知道你在 指定哪一个项目

3. git co origin/Campus_MD —-

a. 切换到 远程库的 Campus_MD 分支

b. [会自动把远程该branch的文件 pull 到本地下来] ??? 未验证这句话

c. origin/ 是远程库的意思

4. git co -b wucanliu_campus_md

a. 创建并切换到自己的本地branch

b. git co XX_campus_md (少了个 -b) ————– 切换到这个 XX branch

5. git push –set-upstream origin wucanliu_campus_md —– 将创建好的新本地 branch 上传到 服务器

6. git add .\app\ —– 让 本地 app 文件夹下的文件 使之得到tracked

7. git commit -a -m “输入更新的内容” ——— 保存修改的 [ 修改明细 ]

8. git push —- 将修改明细上传到远程库
